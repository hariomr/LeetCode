## [22.Generate Parentheses](https://leetcode.com/problems/generate-parentheses/description/)



### Medium

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

## Example 1 :

~~~
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
~~~

### Example 2:

~~~
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
~~~

### Constraints:

- 1 <= n <= 8
